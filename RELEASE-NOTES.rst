============================
 Zenodo-MigrationKit v1.0.0
============================

Zenodo-MigrationKit v1.0.0 was released on TBD, 2015.

About
-----

Module for migration of Zenodo data to Invenio 3.

*This is an experimental developer preview release.*

----------
